ðŸ§  Why is the Nav Bar Vertical?

The navigation bar is a Flexbox container with this rule:

.main-menu ul {
  display: flex;
  flex-direction: column;
}

This tells the browser to stack each <li> vertically inside the <ul>.

Try changing flex-direction to row in the CSS to switch it back to a horizontal layout.
